# **üìå Bootstrap Tables**  

Bootstrap provides **responsive, striped, bordered, hoverable, and dark-themed tables** using built-in classes. Below are different types of tables with examples.

---

## **1Ô∏è‚É£ Basic Table**
Bootstrap applies basic styling to `<table>` elements using the `.table` class.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Table</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>

    <div class="container mt-4">
        <h2>Basic Table</h2>
        <table class="table">
            <thead>
                <tr>
                    <th>#</th>
                    <th>Name</th>
                    <th>Email</th>
                </tr>
            </thead>
            <tbody>
                <tr>
                    <td>1</td>
                    <td>John Doe</td>
                    <td>john@example.com</td>
                </tr>
                <tr>
                    <td>2</td>
                    <td>Jane Smith</td>
                    <td>jane@example.com</td>
                </tr>
            </tbody>
        </table>
    </div>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>


### **üìå Explanation**
| Class | Description |
|---------|-------------|
| `.table` | Adds Bootstrap styling to the table. |
| `<thead>` | Defines the table header section. |
| `<tbody>` | Defines the main body of the table. |

## **2Ô∏è‚É£ Striped Rows**
Use `.table-striped` to add alternating row colors.

### **üìù Code Example**
```html
<table class="table table-striped">
    <thead>
        <tr>
            <th>#</th>
            <th>Name</th>
            <th>Email</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>1</td>
            <td>John Doe</td>
            <td>john@example.com</td>
        </tr>
        <tr>
            <td>2</td>
            <td>Jane Smith</td>
            <td>jane@example.com</td>
        </tr>
    </tbody>
</table>
```

**Effect:** Every alternate row will have a light grey background.

## **6Ô∏è‚É£ Responsive Table**
Use `.table-responsive` to make tables scrollable on small screens.

### **üìù Code Example**
```html
<div class="table-responsive">
    <table class="table table-bordered">
        <thead>
            <tr>
                <th>#</th>
                <th>Name</th>
                <th>Email</th>
                <th>Age</th>
                <th>City</th>
                <th>Country</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>1</td>
                <td>John Doe</td>
                <td>john@example.com</td>
                <td>28</td>
                <td>New York</td>
                <td>USA</td>
            </tr>
        </tbody>
    </table>
</div>
```

**Effect:** On smaller screens, the table will scroll horizontally.
