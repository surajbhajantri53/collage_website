# **🚀 Introduction to Bootstrap**  

## **What is Bootstrap?**  
Bootstrap is a **popular front-end framework** that helps developers build **responsive and mobile-first** websites quickly. It includes **pre-styled components, a grid system, JavaScript plugins, and utility classes** to speed up web development.  

## **🌟 Key Features of Bootstrap**  
✔️ **Responsive Grid System** – Easily create layouts that adapt to different screen sizes.  
✔️ **Pre-styled Components** – Includes buttons, forms, modals, tooltips, and more.  
✔️ **JavaScript Plugins** – Interactive elements like modals, accordions, carousels, etc.  
✔️ **Customizable** – Modify styles using CSS variables or Bootstrap's Sass support.  
✔️ **Cross-browser Compatibility** – Works seamlessly across all modern browsers.  
✔️ **Utility Classes** – Easy-to-use classes for margin, padding, text styles, etc.  

---

## **📌 Bootstrap Setup**  
There are **two ways** to use Bootstrap:  
1. **Via CDN** (No installation needed)  
2. **Via Local Files** (Download Bootstrap and link it manually)  

---

## **1️⃣ Using Bootstrap via CDN (Recommended)**
This is the easiest method—just include the Bootstrap **CSS and JavaScript** files in your HTML.  

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Introduction</title>
    
    <!-- Bootstrap CSS (via CDN) -->
    <!--link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" -->
</head>
<body>
    
    <div class="container text-center mt-5">
        <h1>Welcome to Bootstrap</h1>
        <p class="lead">Bootstrap makes web development faster and easier!</p>
        
        <!-- Bootstrap Button -->
        <button class="btn btn-primary">Click Me</button>
    </div>

    <!-- Bootstrap JavaScript (via CDN) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

</body>
</html>

### **📌 Explanation of Code**
| Element | Description |
|---------|-------------|
| **Bootstrap CSS** | Loads Bootstrap styles via CDN |
| **`.container`** | Adds spacing and centers content |
| **`.text-center`** | Centers text |
| **`.mt-5`** | Adds top margin |
| **`.lead`** | Makes paragraph larger |
| **`.btn btn-primary`** | Adds a Bootstrap button |
| **Bootstrap JS** | Enables JavaScript features like modals & tooltips |

---

## **2️⃣ Using Bootstrap Locally**  
If you want to download Bootstrap and use it **offline**, follow these steps:  
1. **Download Bootstrap** from [getbootstrap.com](https://getbootstrap.com/)  
2. Extract the files and link them in your HTML like this:  

```html
<link rel="stylesheet" href="css/bootstrap.min.css">
<script src="js/bootstrap.bundle.min.js"></script>
```
